MWBar

MWBar is a simple app based on the UCCW apk template. I've created a minimal status bar (minus the notifications) which shows the time, date and battery status. Each of these info areas has a hotspot associated with it to connect the user either with the alarm clock, add calender appointment or view battery statistics. There is a basic grey background and a glass image background version of the bar. This app is compatible only with UCCW. More features will be added in the future. Follow the link below to download from the play store.
